1. 규칙
    1) 2인 대전 게임

    2) 플레이어들은 획득한 코인으로 전략을 짜서 상대편 상대편 끝 20 칸 중 10칸 이상 먹으면 승리

2. 필드 : 20 x 20 2차원배열

3. 코인 
    1) 플레이어들은 턴당 *코인 획득

    2) 플레이어들은 턴마다 코인을 사용하거나 save할 수 있음

    3) save한 코인은 턴 당 10% 이자를 붙여서 다음 턴에 되돌려줌.

4. 병력
    1) 병력은 코인을 사용하여 생산할 수 있다. 

    2) 생산한 병력은 총 5단계로 업그레이드 가능

    3) 병종
        1_ 보병
            농노(낫) -> 징집병 -> 중갑보병 -> 정예보병 -> 프린키페스

        2_ 창병
            농노(쇠스레) -> 징집창병 -> 중갑창병 -> 정예창병 -> 호플리테스 
        
        3_ 기병
            유목민 -> 징집기병 -> 충격기병 -> 정예기병 -> 윙드 후사르

        4_ 영웅
            모든 병력에 +상성

5. 전투
    1) 상성
        1_ 보병 < 기병 < 창병 < 보병
        2_ 영웅은 모든 병종에 상성 우위에 있음
    
    2) 플레이어들은 각자의 병력에 좌표 값을 입력하여 이동가능

    3) 병력의 이동 거리는 병종에 따라 차등
        보병 2, 창병 1, 기병 3

    4) 병력의 hp는 공격하는 상대방 공격력, 자신의 방어력 그리고 상성을 계산하여 적용됨

    5) 1칸에는 유닛 1기만 배치 가능

    6) 턴당 한 행에 있는 모든 유닛이 차례대로 한칸씩 전진하고, 
        만약 그 칸에 상대방의 병력이 있을시 전투, 이기면 한칸 더 전진.

6. 운영
    1) 코인 사용처
        1_ 병력생산
        2_ 병력 업그레이드
        3_ 도박
    
    2) 병력 생산
        1_ 보병, 창병: 1레벨 2코인
        2_ 기병 : 1레벨 3코인
        3_ 영웅: 5레벨 보병, 창병, 기병에서 업그레이드시 
                일정 확률로 보병, 창병, 기병이 영웅으로 바뀜
                확률은 보병,창병 < 기병

    3) 병력 업그레이드
        1_ 보병, 창병 : 레벨 증가할수록 업그레이드 비용 늘어남
        2_ 기병 : 레벨 증가할수록 업그레이드 비용 늘어남
        3_ 영웅 : 영웅은 업그레이드 불가

    4) 도박
        1: 코인
            1_ 빈도 최상 : + 1배
            2_ 빈도 상 : + 1.1배 ~ 1.5배
            3_ 빈도 중 : + 1.6배 ~ 2배 // - 1.1배 ~ 1.5배
            4_ 빈도 하 : + 2.1배 ~ 2.5배 // -1.6배 ~ 1.9배
            5_ 빈도 최하 : +2.6배 ~ 3배 // -2.0배 ~ 2.5배
            6_ 빈도 극악 : +5배
            
        2: 뽑기
            1_ 빈도 최상 : 1레벨 창병, 보병
            2_ 빈도 상 : 2레벨 창병, 보병, 1레벨 기병
            3_ 빈도 중 : 3레벨 창병, 보병, 2레벨 기병
            4_ 빈도 하 : 4레벨 창병, 보병, 3레벨 기병
            5_ 빈도 최하 : 5레벨 창병, 보병, 4레벨 기병
            6_ 빈도 극악 : 영웅 

        3: 신의 선물
            1_ 빈도 극극극악
            2_ 여러가지 게임의 판도를 뒤집을 수 있는 효과
                ex) "고토 회복"         : 1줄 (최대 4칸) 수복 가능
                    "아레스의 축복"     : 턴 종료시 모든 아군 병력에 보너스 능력치
                    "하늘의 분노"       : 턴 종료시 모든 상대 병력 hp -50% (50% 이하 병력은 즉사 판정)
                    "13번째 쥐의 계략"  : 턴 종료시 필드 위 모든 병력 턴당 체력 -20% (새로 생산된 병력에는 적용x) 
                    "바위처럼 단단하게" : 턴 종료시 모든 아군 병력 엄청난 방어력 보너스
                    "헤르메스의 바람"   : 이번 턴 모든 아군 병력 이동 보너스
                    "봉기하는 농노들"   : 턴 종료시 필드 위 모든 1레벨 보병, 창병 엄청난 보너스 능력치
                    "반전된 세계"       : 턴 종료시 필드 위 모든 병력 상성 반전
                    "WRYAAAAAAAA!!!"   : 턴 종료시 필드 위 모든 아군 병력 엄청난 공격력 보너스 
                    "영웅 출현"         : 영웅 3명 생성
